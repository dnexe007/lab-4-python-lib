# Лабораторная работа 3

## 1. Математические функции

Модуль `math_functions.py` содержит реализации различных математических функций:

### Факториал

- **`factorial(n: int) -> int`** - вычисление факториала с помощью цикла

- **`factorial_recursive(n: int) -> int`** - вычисление факториала с помощью рекурсии


**Особенности:**

- Проверка на неотрицательные целые числа

- Исключение `ValueError` при некорректных входных данных


### Числа Фибоначчи

- **`fibonacci(n: int) -> int`** - вычисление n-го члена с использованием формулы Бине

- **`fibonacci_recursive(n: int) -> int`** - вычисление n-го члена с помощью рекурсии


**Особенности:**

- Поддержка больших значений n в формуле Бине

- Рекурсивная версия подходит для небольших значений n


## 2. Алгоритмы сортировки

Модуль `sorts.py` содержит реализации различных алгоритмов сортировки:

### Универсальные сортировки (работают с любыми сравниваемыми типами)

#### Пузырьковая сортировка (`bubble_sort`)

- **Сложность:** O(n²)

- **Память:** O(1)

- **Особенности:** Простая реализация, подходит для небольших массивов


#### Быстрая сортировка (`quick_sort`)

- **Сложность:** O(n log n) в среднем, O(n²) в худшем случае

- **Память:** O(log n)

- **Особенности:** Рекурсивная реализация с выбором опорного элемента случайным образом


#### Пирамидальная сортировка (`heap_sort`)

- **Сложность:** O(n log n)

- **Память:** O(1)

- **Особенности:** Использует структуру данных "куча" (heap)


### Специализированные сортировки

#### Корзинная сортировка (`bucket_sort`)

- **Типы данных:** `int`, `float`

- **Сложность:** O(n + k) в лучшем случае

- **Особенности:** Эффективна при равномерном распределении данных


#### Сортировка подсчетом (`counting_sort`)

- **Типы данных:** `int`

- **Сложность:** O(n + k)

- **Особенности:** Эффективна когда диапазон значений небольшой


#### Поразрядная сортировка (`radix_sort`)

- **Типы данных:** неотрицательные `int`

- **Сложность:** O(dn)

- **Особенности:** Сортировка по разрядам, начиная с младшего


### Общие особенности:

- Все функции возвращают новый отсортированный список (исходный не изменяется)

- Поддержка key-функций для определения порядка сортировки

- Обработка пустых массивов

- Подробные docstrings с описанием параметров


## 3. Реализации стека

Модуль `stacks.py` содержит три различные реализации стека с поддержкой операции нахождения минимума:

### Стек на основе списка (`ListStack`)

- **Внутренняя структура:** список Python

- **Сложность операций:**

    - `push()`: O(1)

    - `pop()`: O(1)

    - `peek()`: O(1)

    - `min()`: O(1)

    - `__len__()`: O(1)

- **Особенности:** Хранит пары (значение, текущий минимум)


### Стек на основе связного списка (`LinkedListStack`)

- **Внутренняя структура:** односвязный список

- **Сложность операций:**

    - `push()`: O(1)

    - `pop()`: O(1)

    - `peek()`: O(1)

    - `min()`: O(1)

    - `__len__()`: O(1) (хранится в отдельном поле)

- **Особенности:** Каждый узел хранит текущий минимум в подстеке


### Стек на основе двух очередей (`QueueStack`)

- **Внутренняя структура:** две очереди (`queue.Queue`)

- **Сложность операций:**

    - `push()`: O(1)

    - `pop()`: O(n)

    - `peek()`: O(n)

    - `min()`: O(1)

    - `__len__()`: O(1) (хранится в отдельном поле)


### Общий функционал всех реализаций:

- **`push(val)`** - добавление элемента

- **`pop()`** - удаление и возврат верхнего элемента

- **`peek()`** - просмотр верхнего элемента без удаления

- **`min()`** - получение минимального элемента в стеке

- **`__len__()`** - получение размера стека

- **Обработка ошибок:** `IndexError` при операциях с пустым стеком

## 4. Тестирование производительности

Модуль `main.py` содержит программу для сравнения производительности алгоритмов сортировки:

### Функция `apply_sort(arr, func)`

- **Назначение:** Применяет функцию сортировки к массиву и выводит результаты

- **Параметры:**

    - `arr`: Исходный массив для сортировки

    - `func`: Функция сортировки для тестирования

- **Функциональность:**

    - Выводит название используемого алгоритма

    - Замеряет время выполнения сортировки в миллисекундах

    - Выводит отсортированный массив

    - Обрабатывает ошибки несовместимости типов данных


### Функция `main()`

- **Назначение:** Основная функция программы для интерактивного тестирования

- **Работа программы:**

    1. Запрашивает у пользователя список чисел через стандартный ввод

    2. Автоматически преобразует числа в целые типы где это возможно

    3. Последовательно тестирует все алгоритмы сортировки:

        - `bubble_sort` - пузырьковая сортировка

        - `quick_sort` - быстрая сортировка

        - `counting_sort` - сортировка подсчетом

        - `radix_sort` - поразрядная сортировка

        - `heap_sort` - пирамидальная сортировка

        - `bucket_sort` - корзинная сортировка

    4. Добавляет сравнение со встроенной функцией `sorted()` Python

    5. Для каждого алгоритма выводит:

        - Отсортированный массив

        - Время выполнения в миллисекундах
